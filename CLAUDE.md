# ペットサロン わんにゃんパラダイス

トリミング・ペットホテルサービスのランディングページ

## 技術スタック

- HTML5 / CSS3 / JavaScript
- Google Fonts (M PLUS Rounded 1c, Kiwi Maru)

## ページ構成

- index.html - トップページ
- reservation.html - 予約ページ

## 特徴

- iOSアプリ風デザイン
- カレンダー形式の予約フォーム
- サービス時間の自動計算機能
- アイボリー・パステルグリーンの配色

## デプロイ

- テンプレート配布用（各クライアントが独自にホスティング）
- ローカルでindex.htmlを開いて確認可能
